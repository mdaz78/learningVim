# Exercise 3: Mixed Usage
1. Open multiple buffers in tabs
2. Split a tab to show multiple buffers
3. Practice commands:
   - :tabs (list tabs)
   - :tabc (close tab)
   - :tabo (close other tabs)
   - :tabm 0 (move tab to start)

Remember:
- :tabnew : create new tab
- :tabc : close current tab
- :tabo : close other tabs
- :tabm N : move tab to position N
- :ls : list buffers
- :b N : go to buffer N
